
# Projet POO NSI
> Dans un but d'apprentissage de la POO (Programation Orientée Objet), la classe d'NSI de Terminale à eu la tâche de créer un projet python utilisant la POO

---

### Idées
- Jeu de cartes
    - Créer un jeu de carte type Balatro ou Solidaire avec des règles spécifiques
- Bac à sable intéractif
    - Créer un bac à sable en 2D avec de la physique (relativement simple) et une forme de vie ayant un comportement en fonction de ce qui l'entoure
- Partition MIDI en algorithmique
    - Créer un générateur de mélodie MIDI (fichier pour sauvegarder des notes de musiques) qui s'adapte en fonction de règles de la théorie musicale
